# 🚀 Lotto 6/45 Generator Pro - 향후 기능 제안서

현재 프로젝트(v2.5)의 문서(`README.md`, `claude.md`, `gemini.md`)를 분석한 결과를 바탕으로, 다음 버전에서 추가하면 좋을 기능들을 제안합니다.

## 1. 🧠 AI 기반 번호 예측 (AI Prediction Mode) - **(v2.6 완료)**
*   **개요**: 단순 통계(Hot/Cold)를 넘어 머신러닝(LSTM 또는 회귀 분석) 모델을 활용하여 다음 회차 번호를 예측하는 기능입니다.
*   **구현 방안**: 
    *   `scikit-learn` 또는 `TensorFlow Lite`를 사용하여 최근 100회차 패턴 학습
    *   사용자에게 "AI 추천 번호" 별도 탭 제공
*   **기대 효과**: 기술적으로 진보된 느낌을 주며, 단순 무작위 생성보다 사용자 신뢰도를 높일 수 있습니다.

## 2. 🧪 과거 시뮬레이션 (Backtesting System) - **(v2.6 완료)**
*   **개요**: 사용자가 선택한 생성 전략(예: 홀짝 비율 3:3, 합계 120~150)을 과거 1년치 데이터에 적용했을 때의 가상 수익률을 계산해주는 기능입니다.
*   **구현 방안**:
    *   기존 `lotto_history.db` 데이터를 활용하여 과거 회차에 대입
    *   "만약 이 전략으로 1년간 매주 5천원씩 샀다면?" 결과 리포트 제공
*   **기대 효과**: 사용자가 자신만의 전략을 검증하는 재미를 느낄 수 있습니다.

## 3. ☁️ 클라우드 동기화 (Cloud Sync)
*   **개요**: `favorites.json`, `history.json` 파일을 Google Drive 또는 개인 클라우드와 동기화하는 기능입니다.
*   **구현 방안**:
    *   간단한 구글 로그인 연동 또는 내보내기/가져오기 기능 강화
    *   설정 파일(`.lotto_generator/`)의 백업/복원 원클릭 솔루션
*   **기대 효과**: PC 포맷이나 기기 이동 시 데이터 유실을 방지합니다.

## 4. 🔔 당첨 자동 알림 (Notification Service)
*   **개요**: 매주 토요일 추첨 방송 직후, 사용자가 생성해둔 번호 중 당첨 번호가 있는지 자동으로 확인하고 알림을 줍니다.
*   **구현 방안**:
    *   시스템 트레이 아이콘으로 백그라운드 실행
    *   추첨 시간(토요일 20:35) 이후 자동 API 조회 및 팝업 알림
*   **기대 효과**: 사용자가 매번 번호를 확인하러 들어올 필요 없이 능동적으로 결과를 알려줍니다.

## 5. 🗺️ 고급 시각화 대시보드 (Advanced Dashboard)
*   **개요**: 현재의 텍스트/기본 차트 위주 통계를 더 직관적인 히트맵(Heatmap)이나 그래프로 고도화합니다.
*   **구현 방안**:
    *   **번호별 출현 빈도 히트맵**: 최근 5주/10주/20주 기간별 색상 농도로 표시
    *   **미출현 번호 타임라인**: 특정 번호가 마지막으로 나온 지 얼마나 되었는지 시각화
*   **기대 효과**: 데이터 분석을 좋아하는 사용자들에게 강력한 도구를 제공합니다.
