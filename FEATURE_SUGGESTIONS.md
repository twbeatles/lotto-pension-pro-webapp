# 🚀 로또 6/45 프로 향후 기능 제안서

현재 문서(`README.md`, `claude.md`, `gemini.md`)와 구현 흐름을 기준으로, 다음 버전에서 고려할 만한 기능을 정리했습니다.

## 1. 🧠 인공지능 예측 고도화 - **(v2.6 완료)**
- **개요**: 단순 고빈도/저빈도 분석을 넘어 학습 기반 예측 정확도를 높이는 기능
- **구현 방향**:
  - `scikit-learn` 또는 `TensorFlow Lite`로 최근 회차 패턴 학습
  - 예측 모델별 결과 비교 화면 강화
- **기대 효과**: 차별화된 예측 경험 제공, 전략 선택 신뢰도 향상

## 2. 🧪 과거 전략 시뮬레이션 강화 - **(v2.6 완료)**
- **개요**: 사용자가 설정한 전략을 과거 데이터에 대입해 성과를 검증
- **구현 방향**:
  - 기간/구매량/전략별 수익률·당첨률 자동 비교
  - "이 전략을 1년간 유지했을 때" 같은 시나리오 보고서 제공
- **기대 효과**: 사용자 전략 검증 재미와 체류 시간 증가

## 3. ☁️ 클라우드 동기화
- **개요**: 즐겨찾기/히스토리/티켓북 데이터를 계정 기반으로 동기화
- **구현 방향**:
  - 구글 로그인 기반 동기화 또는 외부 저장소 연동
  - 백업/복원 과정을 더 단순한 한 번 클릭 방식으로 개선
- **기대 효과**: 기기 변경·재설치 시 데이터 유실 최소화

## 4. 🔔 당첨 자동 알림
- **개요**: 추첨 이후 저장된 티켓의 당첨 여부를 자동 확인해 알림 제공
- **구현 방향**:
  - 추첨 시간 이후 자동 조회 예약
  - 인앱 알림/시스템 알림 채널 분리 제공
- **기대 효과**: 사용자가 매번 수동 확인하지 않아도 결과 확인 가능

## 5. 🗺️ 고급 시각화 대시보드
- **개요**: 현재 통계 화면을 더 직관적인 시각 자료 중심으로 확장
- **구현 방향**:
  - 기간별 번호 출현 열지도
  - 번호별 미출현 기간 타임라인
- **기대 효과**: 데이터 분석 성향 사용자에게 높은 활용도 제공
