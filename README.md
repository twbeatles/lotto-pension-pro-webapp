# 로또·연금복권 프로

동행복권 **로또 6/45**와 **연금복권720+**의 당첨 통계를 분석하고, 번호를 생성·저장·관리하는 무료 웹앱입니다.

**▶ 앱 바로가기:** https://twbeatles.github.io/lotto-pension-pro-webapp/

> 모든 번호 추천은 과거 통계 기반 참고용입니다. 당첨을 보장하지 않습니다.

---

## 주요 기능

| 탭 | 기능 요약 |
|---|---|
| **번호 생성** | 조건(고정수·제외수·연속수)을 설정해 로또 6/45 번호 조합 생성, QR 코드 출력 |
| **당첨 통계** | 역대 당첨 번호 출현 빈도·분포 분석 |
| **번호 추천** | 다중 통계 신호(빈도·최근성·추세·페어 등) 기반 번호 추천, 분석 강도 조절 |
| **연금복권** | 연금복권720+ 통계·추천·저장·당첨 확인·CSV 내보내기 |
| **시뮬레이션** | 로또 6/45 전략 백테스트 |
| **당첨 확인** | 저장한 번호와 최신 당첨 결과 자동 비교 |
| **데이터 관리** | 전체 저장 데이터 내보내기·가져오기 (백업/복원) |

## 사용 방법

### 번호 생성 (로또 6/45)

1. **번호 생성** 탭을 엽니다.
2. `번호 생성하기`를 클릭해 번호 조합을 만듭니다.
   - **고정수**: 반드시 포함할 번호를 지정합니다.
   - **제외수**: 포함하지 않을 번호를 지정합니다.
   - **연속수 제한**: 연속된 번호를 최대 몇 개까지 허용할지 설정합니다.
3. 마음에 드는 조합을 **내 번호 보관함**에 저장합니다.
4. QR 코드를 생성해 복권 구매 매장에서 활용할 수 있습니다.

### 번호 추천 (로또 6/45)

1. **번호 추천** 탭을 엽니다.
2. 원하는 **전략**을 선택하거나 `자동 선택`을 사용합니다.
3. **분석 강도**를 고릅니다. 처음이라면 `기본`을 권장합니다.
4. `추천 시작`을 눌러 결과를 받습니다.
5. 추천 근거(빈도, 최근성, 추세 등)를 확인하고, 마음에 드는 번호를 저장합니다.

> 같은 번호를 다시 만들고 싶다면 결과 화면의 **재현 코드**를 저장해두세요.

### 연금복권720+ 추천

1. **연금복권** 탭을 엽니다.
2. 전략을 선택합니다.
   - 혼합 균형, 자리별 강세, 끝자리 적중형, 조 로테이션, 공백 반등, 보너스 흐름, 완전 랜덤 등
3. 원하는 필터를 설정합니다. (조 선택, 자리 고정/제외, 숫자 합계, 홀수 비율 등)
4. `추천 시작`을 눌러 번호를 생성합니다.
5. 번호를 개별 저장하거나, 같은 6자리로 여러 조를 **확장 저장**할 수 있습니다.

### 당첨 확인

- **로또 6/45**: **당첨 확인** 탭에서 저장된 번호를 최신 회차 당첨 번호와 자동 비교합니다.
- **연금복권720+**: **연금복권** 탭 저장 목록에서 대상 회차 우선 확인 또는 최신 회차 참고 비교를 합니다.

### 캠페인

여러 회차에 걸쳐 같은 번호를 추적할 때 **캠페인**을 활용합니다.

- 로또 6/45와 연금복권720+ 모두 별도 캠페인을 지원합니다.
- 시작 회차와 회차 수를 지정해 캠페인을 만들고, 회차별 당첨 결과를 한 번에 확인할 수 있습니다.

### 데이터 백업 및 복원

저장한 번호·캠페인·설정을 파일로 백업할 수 있습니다.

**백업하기**
1. **데이터 관리** 탭 > `전체 데이터 내보내기`
2. JSON 백업 파일이 다운로드됩니다.

**복원하기**
1. **데이터 관리** 탭 > `가져오기`
2. 저장해둔 백업 JSON 파일을 선택합니다.
3. 덮어쓰기 전에 자동으로 안전 백업이 먼저 생성됩니다.

> 백업 파일은 최대 32 MB까지 가져올 수 있습니다.

## 앱 설치 (PWA)

스마트폰이나 PC 브라우저에서 홈 화면에 설치해 네이티브 앱처럼 사용할 수 있습니다.

- **Android / Chrome**: 주소창 오른쪽 `설치` 아이콘 탭
- **iOS / Safari**: 하단 공유 버튼 → `홈 화면에 추가`
- **PC / Chrome · Edge**: 주소창 오른쪽 설치 아이콘 클릭

설치된 앱은 오프라인 상태에서도 저장된 데이터와 기본 기능을 사용할 수 있습니다.

## 자주 묻는 질문

**Q. 번호 추천이 당첨을 보장하나요?**
아닙니다. 모든 추천은 과거 통계 기반 참고용이며, 로또 추첨은 완전한 무작위입니다.

**Q. 저장한 데이터가 사라졌어요.**
앱 데이터는 브라우저 로컬 저장소에 보관됩니다. 브라우저의 쿠키·사이트 데이터를 전체 초기화하면 함께 삭제됩니다. **데이터 관리 > 전체 데이터 내보내기**로 정기 백업을 권장합니다.

**Q. 여러 기기에서 같은 데이터를 쓰고 싶어요.**
기기 간 자동 동기화는 지원하지 않습니다. **데이터 내보내기**로 파일을 저장한 뒤, 다른 기기에서 **가져오기**로 복원하세요.

**Q. 오프라인에서도 사용할 수 있나요?**
PWA로 설치하면 오프라인 상태에서도 저장된 데이터와 기본 기능을 사용할 수 있습니다.

## 고급: 데이터 연결 주소

최신 로또 당첨 번호를 더 빠르게 동기화하려면 Cloudflare Worker 프록시를 직접 배포하고
**설정 > 데이터 연결 주소(고급)**에 입력할 수 있습니다.
설정하지 않아도 기본 자동 동기화가 작동합니다.

→ 프록시 배포 방법: [proxy/README.md](proxy/README.md)
